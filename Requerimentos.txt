O sistema pode ser utilizado numa máquina Windows 11 Pro x64 ou Ubuntu 20.04 lts.
Foi utilizada a linguagem Python versão 3.10.7 no Windows 11 e Python 3.8.10 no Ubuntu.
Utilizamos os módulos socket para comunicação de cliente servidor e o datetime para utilizar as datas.
Também utilizamos o módulo "pika", esse módulo precisa ser instalado no pip, utilizando "pip install pika"
e esse módulo também precisa estar numa versão maior ou igual a 1.1.0.
